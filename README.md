 使用ツール： VisualStudio / C++ / OpenGL

#### 説明</br>
人の断層撮影イメージを合わせて３Dモデルを作り出す RayCastingプログラムです。</br>
３Dモデルの皮膚や皮膚の中にある骨を見られる機能とモデルを回転しながら見られる機能があります。</br>
照明モデルはPhong公式を利用しました。
